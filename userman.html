Return Policy

  1. Return Period:
    • Return requests are only accepted within 24 hours after purchasing the service.
    • Return requests made after 24 hours will not be considered.
  2. Return Conditions:
    • Return requests are considered when there is an issue related to the quality, functionality, or promised features of the service.
    • Return requests made after starting to use or activate the service will not be accepted.
  3. Return Request:
    • You can submit your return request by contacting our customer service through the communication channels on the ellusion.shop website (phone, email, contact form).
    • During the return request, you need to clearly state your purchase information, the date of the service, and the reason for your return request.
  4. Return Process:
    • After reviewing your return request, the status of the service and other factors related to your request will be evaluated.
    • When a valid return request is determined, the return process will be initiated, and your payment will be refunded as soon as possible.
    • The completion of the return process may vary depending on transaction volume and your payment provider.
  5. Return Denial:
    • Return requests may be denied under the following circumstances:
      • Requests made outside the 24-hour return period.
      • Return requests that do not comply with the return conditions mentioned above.
      • Return requests made after the service has been used or activated.

Please note that our return policy is only applicable under specific conditions. It is important to follow the specified timeframes and conditions for return requests. Our customer service team will be happy to assist you with any return request.

This return policy has been set by ellusion.shop and you can refer to our policy for updates and changes. The acceptance or denial of return requests will be subject to the aforementioned terms and conditions.

After completing your return request, we will cease processing your personal data in accordance with legal requirements. Once the return process is completed, your personal data may be deleted or anonymized in compliance with legal retention periods. Anonymized data may be used for purposes such as statistical analysis or research, but you will not be personally identifiable.

The data processing period is limited to the completion of your return request and the finalization of the return process. We take necessary technical and administrative measures to ensure the security of your personal data during this process.

In compliance with the Personal Data Protection Law, we collect and use your personal data only for specific purposes. When processing your personal data related to return requests, we only use the necessary information to carry out the return process and do not retain this information unnecessarily.

Please remember that you have rights regarding the collection, use, and processing of your personal data in accordance with the Personal Data Protection Law.
